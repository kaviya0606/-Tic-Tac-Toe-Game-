# Tic Tac Toe Game in C++

## Project Overview

This project is a console-based Tic Tac Toe game developed using C++. The game allows two players to play against each other in a turn-based environment. It demonstrates core programming concepts such as arrays, loops, conditional statements, functions, and game logic implementation.

## Features

* Two-player gameplay (X and O)
* Dynamic board display after every move
* Win detection using rows, columns, and diagonals
* Draw detection
* Input validation for invalid moves
* Replay option to start a new game
* Simple console-based interface

## Technologies Used

* C++
* Arrays
* Functions
* Loops
* Conditional Statements
* Console I/O

## How to Run

1. Open the project in Code::Blocks or any C++ IDE.
2. Compile the `main.cpp` file.
3. Run the program.
4. Follow the on-screen instructions to play the game.

## Learning Outcomes

Through this project, the following concepts are demonstrated:

* Array manipulation
* Function-based programming
* User input handling
* Conditional logic
* Game state management
* Problem-solving using C++
